# Arrow Contracts

Implementation of the Arrow token and governance smart contracts.

## Get Started

```bash
npm install
```

## Testing

```bash
npm run test
```

## Deployments

To deploy on Optimism Kovan testnet:

```bash
npm run deployOptimismKovan
```
